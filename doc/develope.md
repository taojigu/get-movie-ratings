
when the extension ui is presented, the current page (like movie-collection.html) is read and parsed. The movies inside the pages are listed with list item.
Inside every list item, the name and image of movie is displayed.there is a button which links to the douban search page with the movie title as keyword. The extension does not collect or transmit any data, and all processing is done locally in the user's browser.
When user clicked the open, the extension ui will navigate to the douban search page.
On the top of the page, there is a navigation bar, with which the user can switch between the different pages.
